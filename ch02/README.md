# Chapter 2. 메타데이터 설계 실습

> 🚧 본 챕터의 자료는 집필 중입니다. 출간 전까지 단계적으로 추가됩니다.

## 📁 예정 자료

- `ch02_metadata_schema.ipynb` — Pandas로 스키마 정의 및 검증
- `ch02_dublin_core_sample.csv` — Dublin Core 15요소 채워 보기 샘플

## 🎯 학습 목표

- Dublin Core 15요소를 활용해 기본 메타데이터를 작성한다
- AI 활용을 위한 확장 필드(요약·키워드·임베딩ID·소스)를 설계한다
- 우리 기관용 메타데이터 스키마를 CSV로 정의·검증한다
