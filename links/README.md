# /links — 인용 URL 아카이브

본서에서 인용한 모든 URL의 **출간 시점 스냅숏**과 **사후 보존 상태**를 관리합니다.

## 운영 원칙

1. 본서 인용 URL은 출간 직전 일괄 재검증한다.
2. 사라진 URL은 [Wayback Machine](https://web.archive.org/) 스냅숏 주소를 [`archive.md`](./archive.md)에 등록한다.
3. 새 스냅숏이 필요하면 [Wayback Machine "Save Page Now"](https://web.archive.org/save) 페이지에서 즉시 저장한 후 등록한다.

## 파일

- [`archive.md`](./archive.md) — 챕터별 URL 보존 상태표
