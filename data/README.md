# /data — 공통 샘플 데이터셋

여러 챕터에서 공통으로 쓰는 샘플 데이터를 보관합니다.

## 예정 데이터

| 파일 | 내용 | 라이선스 |
|------|------|----------|
| `sample_books.csv` | 도서 메타데이터 50건 (가상) | CC0 |
| `sample_articles.jsonl` | 학술 초록 30건 (공개) | CC BY |
| `sample_pdfs/` | 실습용 공공 PDF 5종 | 공공누리 1유형 |

## 데이터 사용 원칙

- 모든 샘플 데이터는 **교육 목적의 무상 사용**이 허용된 자료만 포함합니다.
- 실제 운영 데이터·개인정보·미공개 자료는 절대 업로드하지 않습니다.
- 각 데이터셋의 원 출처와 라이선스는 해당 디렉터리 내 `SOURCES.md`에 명시합니다.
